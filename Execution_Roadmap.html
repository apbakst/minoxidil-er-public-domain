Phase 1: Formulation Development

Duration: 2-4 weeks | Cost: $2,000-5,000

1.1 Make Test Batches

  • Source excipients (Compritol, Precirol, PEG 6000, Poloxamer 188, MCC, lactose)
  • Source minoxidil API (USP grade)
  • Make 3 small batches (50-100 units each) per the formulation protocol
  • Document batch records for each
  • Visual inspection: tablets/capsules uniform, no defects

1.2 Dissolution Testing (Critical)

  • Run dissolution on each batch (1, 4, 8, 12 hours)
  • Compare to target: 15-20% (1h), 40-50% (4h), 65-75% (8h), >80% (12h)
  • If too fast: increase Compritol, decrease PEG
  • If too slow: decrease Compritol, increase PEG
  • Iterate until dissolution meets target (may need 2-4 iterations)

1.3 Fed-State Dissolution

  • Run dissolution in FeSSIF or milk (simulates fed state)
  • Compare to fasted state dissolution
  • Acceptable if fed-state 4h dissolution is <1.5x fasted
  • If food effect too high: add calcium stearate or carnauba wax
Phase 1 Exit Criteria:
  • Dissolution meets target profile in fasted media
  • Food effect is acceptable (<1.5x at 4h)
  • Batch-to-batch variability <15%

Phase 2: Analytical & Quality Control

Duration: 2-3 weeks | Cost: $3,000-8,000

2.1 Assay Method

  • HPLC method for minoxidil content (or use compendial USP method)
  • Verify method works with your matrix (lipid interference check)
  • Run assay on 3 batches: target 95-105% of label claim

2.2 Content Uniformity

  • Test 10 individual units from each batch
  • Calculate AV (Acceptance Value) per USP <905>
  • Target: AV ≤ 15.0

2.3 Physical Tests

  • Weight variation (±5%)
  • Hardness (tablets): 6-10 kP
  • Friability (tablets): <1.0%
  • Disintegration: should NOT disintegrate in <1 hour (it's ER)
Phase 2 Exit Criteria:
  • Assay: 95-105% of label claim
  • Content uniformity: AV ≤ 15.0
  • Physical tests pass

Phase 3: Stability Testing

Duration: Ongoing (1-6 months) | Cost: $2,000-5,000

3.1 Accelerated Stability

  • Place samples at 40°C / 75% RH
  • Test at: 1 week, 2 weeks, 1 month, 3 months, 6 months
  • Test: Appearance, Assay, Dissolution, DSC (lipid form)

3.2 Real-Time Stability

  • Place samples at 25°C / 60% RH
  • Test at: 1 month, 3 months, 6 months, 12 months
  • This runs in parallel - don't wait for completion

3.3 DSC/XRD Monitoring (Polymorphism)

  • Run DSC at T0 and each stability timepoint
  • Watch for peak shifts indicating polymorphic transition
  • If dissolution slows >15% at 1 month accelerated: investigate
Minimum Before Human Use:
  • 1-month accelerated stability data with no significant changes
  • Dissolution within ±10% of initial
  • No assay drop >5%

You can proceed to human use after 1-month accelerated data while continuing longer stability studies.

Phase 4: Animal PK Study (Recommended but Optional)

Duration: 4-6 weeks | Cost: $30,000-50,000

4.1 Study Design

  • Species: Beagle dogs (n=6)
  • Design: 3-way crossover (ER fasted, ER fed, IR solution)
  • Dose: ~2-3 mg/kg (approximately 20-30 mg per dog)
  • Washout: 1 week between periods

4.2 Sampling & Analysis

  • Blood samples: pre-dose, 0.5, 1, 2, 3, 4, 6, 8, 12, 24h
  • Plasma separation and storage at -80°C
  • LC-MS/MS analysis for minoxidil
  • Calculate: Cmax, Tmax, AUC0-24, food effect ratio

4.3 Success Criteria

  • Tmax delayed ≥2 hours vs IR
  • Cmax reduced ≥25% vs IR
  • AUC ≥70% of IR (bioavailability acceptable)
  • Food effect <2-fold on Cmax
When to Skip Animal PK:
  • If going compounding-only route (503A)
  • If dissolution data looks excellent
  • If budget is limited - can do human PK directly (with physician oversight)

Animal PK de-risks but isn't strictly required for compounding.

Phase 5: Compounding Setup

Duration: 2-4 weeks | Cost: $1,000-3,000

5.1 Pharmacy Selection

  • Identify 2-3 candidate compounding pharmacies
  • Confirm lipid matrix / melt granulation capability
  • Confirm 503A or 503B registration
  • Request references / quality certifications
  • Select pharmacy partner

5.2 Technology Transfer

  • Provide formulation protocol to pharmacy
  • Review process with their pharmacist
  • Pharmacy makes qualification batch
  • Run dissolution on pharmacy's batch
  • Confirm results match your development batches

5.3 Prescriber Setup

  • Identify 1-3 dermatologists willing to prescribe off-label
  • Provide them with: formulation info, dosing rationale, safety data
  • Establish prescription workflow with pharmacy
  • Create patient consent/information document
Phase 5 Exit Criteria:
  • Pharmacy can successfully compound the formulation
  • Pharmacy's batch matches your dissolution profile
  • At least one prescribing physician ready
  • Patient information materials ready

Master Checklist: Before Human Use

  • Content uniformity: AV ≤ 15.0
  • 1-month accelerated stability: no significant change
  • Compounding pharmacy can reproduce your results
  • Licensed physician willing to prescribe

Key Decision Points

Decision 1: After Initial Dissolution Testing

  • If dissolution meets spec: Proceed to QC and stability
  • If dissolution too fast: Increase Compritol, decrease PEG, re-test
  • If dissolution too slow: Decrease Compritol, increase PEG, re-test
  • If can't get it right after 4 iterations: Consider pivoting to MUPS

Decision 2: After 1-Month Accelerated Stability

  • If stable: Proceed to compounding setup
  • If dissolution slows >15%: Add annealing step, re-test
  • If assay drops >5%: Investigate degradation, may need formulation change

Decision 3: Dog PK Study?

  • Do it if: You plan to eventually seek FDA approval, or want to de-risk
  • Skip it if: Budget limited, going compounding-only, dissolution looks excellent

Executive Summary

Objective: Design extended-release oral minoxidil formulations that avoid Veradermics' patent claims (WO2024091572A1 / US12268688B2) which specifically cover HPMC-based matrix tablets.

This document presents five alternative formulation approaches using fundamentally different release mechanisms:

  1. Elementary Osmotic Pump (EOP) - Osmotic pressure-driven release through laser-drilled orifice
  2. Controlled Porosity Osmotic Pump (CPOP) - Porous membrane osmotic system
  3. Multi-Particulate Pellet System (MUPS) - Ethylcellulose-coated pellets in capsule
  4. Lipid Matrix Tablet - Glyceryl behenate/palmitostearate waxy matrix
  5. Bi-Layer Push-Pull Osmotic Tablet - Two-compartment mechanical pump

1 Elementary Osmotic Pump Tablet (EOP)

Mechanism

Osmotic pressure drives drug solution through laser-drilled orifice. Uses cellulose acetate semipermeable membrane. Fundamentally different from HPMC matrix swelling/erosion.

Povidone K30 16.5 6.6% Binder
Magnesium stearate 6.25 2.5% Lubricant
Colloidal silicon dioxide 6.25 2.5% Glidant
Total core weight 250.0 100%

Semipermeable Membrane Coating

Component Amount Function
Cellulose acetate (398-10) 85% of coating Semipermeable membrane former
PEG 3350 15% of coating Plasticizer/pore former
Coating weight gain 8-10% Controls water permeation
Orifice diameter 0.5 mm Laser-drilled delivery port

Expected Release Profile

  • Lag time: 1-2 hours (membrane hydration)
  • Zero-order release: 2-12 hours
  • >85% release by 14 hours
Key Patent Differentiation: No HPMC used. Osmotic mechanism vs. gel diffusion. Cellulose acetate membrane vs. HPMC matrix. Requires laser-drilled orifice.

RECOMMENDED 3 Multi-Particulate Pellet System (MUPS)

Mechanism

Drug-loaded pellets with rate-controlling ethylcellulose/Eudragit polymer coating in capsule. Completely different architecture from monolithic matrix tablet.

Pellet Core (drug-layered on sugar spheres)

Component Amount per capsule Function
Sugar spheres (25-30 mesh) 200 mg Inert core
Minoxidil 8.5 mg Active ingredient
HPMC E5 (binder solution) 15 mg Drug layering binder
Talc 8 mg Anti-tacking agent

Sustained-Release Coating

Component % of coating Function
Ethylcellulose (Surelease®) 60% Rate-controlling polymer
Eudragit® RS 30D 30% Sustained release modifier
Triethyl citrate 10% Plasticizer
Coating weight gain 15-20% Controls release rate

Capsule Fill

Component Amount
Coated pellets ~290 mg
Capsule Size 1 hard gelatin or HPMC

Expected Release Profile

  • 20-30% release at 2 hours
  • 50-60% release at 6 hours
  • >85% release at 12 hours
Key Patent Differentiation: Multi-particulate vs. monolithic. Ethylcellulose-based coating vs. HPMC matrix. Capsule dosage form. Diffusion through coating vs. gel erosion.
Why This Is Recommended:
  • Most established regulatory pathway for ER generics
  • No specialized equipment (laser drilling) required
  • Flexible release profile customization
  • Lower manufacturing complexity
  • Clear patent differentiation
  • Reduced dose-dumping risk

2ND CHOICE 4 Lipid Matrix Tablet

Mechanism

Drug dispersed in waxy lipid matrix. Release by erosion and diffusion through hydrophobic lipid. Fundamentally different from hydrophilic HPMC gel matrix.

Tablet Composition (per tablet)

Component Amount (mg) % w/w Function
Minoxidil 8.5 3.4% Active ingredient
Compritol® 888 ATO (glyceryl behenate) 100.0 40.0% Lipid matrix former
Precirol® ATO 5 (glyceryl palmitostearate) 50.0 20.0% Secondary lipid
Microcrystalline cellulose 62.5 25.0% Filler
Lactose monohydrate 17.0 6.8% Filler
Colloidal silicon dioxide 6.0 2.4% Glidant
Magnesium stearate 6.0 2.4% Lubricant
Total tablet weight 250.0 100%

Manufacturing Process (Melt Granulation)

  1. Preheat mixer to 75-80°C
  2. Add Compritol + Precirol, melt completely
  3. Add minoxidil, mix until dispersed
  4. Add fillers, mix thoroughly
  5. Cool and solidify
  6. Mill through 20-mesh screen
  7. Blend with lubricant/glidant
  8. Direct compress into tablets
Key Patent Differentiation: Lipid-based vs. hydrophilic polymer. Hydrophobic matrix. Drug release via lipid erosion. pH-independent release. No polymers in matrix.

5 Bi-Layer Push-Pull Osmotic Tablet

Development Recommendations

First Choice: Formulation 3 (MUPS)

  • Most established regulatory pathway
  • No specialized equipment required
  • Ethylcellulose coatings well-characterized
  • Flexible release customization
  • Clear patent differentiation

Second Choice: Formulation 4 (Lipid Matrix)

  • Simple manufacturing (melt granulation)
  • Completely different chemistry
  • Strong patent differentiation
  • pH-independent release
  • Low-cost excipients

Important Caveats

  1. Patent freedom-to-operate analysis required - Have IP attorney review against all Veradermics claims
  2. Formulation optimization needed - These are starting points requiring development
  3. Bioequivalence studies required - Different mechanisms may yield different PK profiles
  4. Stability studies - Minoxidil compatibility with each matrix type needs verification
  5. Clinical PK studies - Required to confirm target profile achieved
